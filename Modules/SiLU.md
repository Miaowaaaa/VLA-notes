
# 深度学习核心组件知识图谱

## 一、 nn.SiLU() 激活函数（Swish）

`nn.SiLU()` 是现代深度学习模型（如 YOLOv5/v8、EfficientNet）中非常热门的激活函数。

### 1. 结构与公式

* **数学表达式：** 
$$f(x) = x \cdot \sigma(x) = \frac{x}{1 + e^{-x}}$$


* **结构本质（自门控机制）：** 输入 $x$ 分为两路：一路保持原样，另一路经过 Sigmoid 函数 $\sigma(x)$ 转化为 $0 \sim 1$ 的概率门控值。最终将两路结果进行**按元素相乘（Element-wise Multiplication）**，由输入自己决定通过多少信息。

### 2. 核心数学特性

* **无上界、有下界：** 当 $x \to +\infty$ 时，输出接近 $x$；当 $x \to -\infty$ 时，输出趋于 $0$。
* **平滑非单调性：** 全定义域内**处处可导**。在 $x < 0$ 的区域有一个微小的下凹（并非像 ReLU 那样在 0 处直接死板折断）。
* **极小值点的推导：** 通过对 $f(x)$ 求导并令 $f'(x) = 0$，得到超越方程 $(1 + x)e^{-(1+x)} = -e^{-1}$。利用 **Lambert W 函数** 求解，得出：
* **极小值点：** $x \approx -1.28$
* **全局最小值：** $f(-1.28) \approx \mathbf{-0.28}$（精准值 $-0.27846$）


* **优势：** 处处可导的平滑特性使得深层网络在梯度下降时**反向传播更稳定，模型收敛更快**。

---


以下是完整的数学推导过程：

### 1. 建立函数与求导

SiLU 函数的表达式为：


$$f(x) = x \cdot \sigma(x) = \frac{x}{1 + e^{-x}}$$

我们要找到它的最小值，需要先对 $x$ 求导。利用乘法求导法则 $(uv)' = u'v + uv'$，以及 Sigmoid 函数的导数特性 $\sigma'(x) = \sigma(x)(1 - \sigma(x))$：

$$f'(x) = 1 \cdot \sigma(x) + x \cdot \sigma'(x)$$

$$f'(x) = \sigma(x) + x \cdot \sigma(x)(1 - \sigma(x))$$

$$f'(x) = \sigma(x) \big( 1 + x(1 - \sigma(x)) \big)$$

---

### 2. 令导数为 0 求解临界点

为了找到极值点，令 $f'(x) = 0$。
因为对于任意实数 $x$，$\sigma(x) > 0$ 恒成立，所以只能是括号内的部分为 0：

$$1 + x(1 - \sigma(x)) = 0$$

将 $\sigma(x) = \frac{1}{1 + e^{-x}}$ 代入上式：


$$1 + x \left( 1 - \frac{1}{1 + e^{-x}} \right) = 0$$

$$1 + x \left( \frac{e^{-x}}{1 + e^{-x}} \right) = 0$$

两边同乘以 $1 + e^{-x}$：


$$1 + e^{-x} + x e^{-x} = 0$$

$$1 + (1 + x)e^{-x} = 0$$

$$(1 + x)e^{-x} = -1$$

$$(1 + x)e^{-(1+x)} = -e^{-1}$$

这是一个经典的超越方程，需要使用 **Lambert W 函数**（$W$ 函数，满足 $W(z)e^{W(z)} = z$）来求解：


$$1 + x = W_{-1}(-e^{-1})$$

$$x = W_{-1}(-e^{-1}) - 1$$

通过数值计算可以得到该点的精确近似值：


$$x \approx -1.27846$$

---

### 3. 计算极小值

将求得的极小值点 $x \approx -1.27846$ 代入原函数 $f(x)$ 中：

$$f(-1.27846) = \frac{-1.27846}{1 + e^{-(-1.27846)}} = \frac{-1.27846}{1 + e^{1.27846}}$$

计算分母：


$$e^{1.27846} \approx 3.59114$$

$$1 + 3.59114 = 4.59114$$

最终算出函数值：


$$f(-1.27846) \approx \frac{-1.27846}{4.59114} \approx -0.27846$$

---

### 结论

经过精确的微积分计算，SiLU 函数在 $x \approx -1.28$ 处取得全局最小值，其对应的函数值约为 **$-0.27846$**（通常四舍五入保留两位小数，即 **$-0.28$**）。
